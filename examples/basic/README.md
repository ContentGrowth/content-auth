# Basic Example

This example demonstrates how to use `@contentgrowth/content-auth` with Hono and Cloudflare Workers.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open http://localhost:5173

## Notes

- This example uses a mock D1 database in development.
- The `BETTER_AUTH_SECRET` is set in `wrangler.toml`.
